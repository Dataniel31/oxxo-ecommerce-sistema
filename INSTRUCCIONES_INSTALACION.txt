📋 REQUISITOS PREVIOS
===================
✅ Node.js (versión 16 o superior)
   - Descargar desde: https://nodejs.org/
   - Verificar instalación: node --version

✅ Navegador web moderno (Chrome, Firefox, Edge, Safari)

✅ Dispositivo con cámara (para scanner de códigos QR/barras)


📦 INSTALACIÓN PASO A PASO
==========================

1️⃣ OBTENER EL CÓDIGO
   - Si tienes ZIP: Descomprimir en carpeta deseada

2️⃣ NAVEGAR A LA CARPETA
   - Abrir terminal/CMD
   - cd Demo-MVP
   - Verificar que existe package.json

3️⃣ INSTALAR DEPENDENCIAS
   npm install
   
   📝 Nota: Esto descargará todas las librerías necesarias
   ⏱️ Tiempo estimado: 2-5 minutos

4️⃣ VERIFICAR INSTALACIÓN
   - Debe aparecer carpeta "node_modules"
   - No debe haber errores en terminal


🚀 EJECUTAR EL PROGRAMA
======================

1️⃣ MODO DESARROLLO (Recomendado para pruebas)
   npm run dev
   
   ✅ URLs disponibles:
   - Local: https://localhost:5175/
   - Red: https://[TU_IP]:5175/
   
   📱 Para móvil: Usar la URL de red (ej: https://192.168.1.100:5175/)

2️⃣ MODO PRODUCCIÓN (Para uso final)
   npm run build
   npm run preview

⚠️ SOLUCIÓN DE PROBLEMAS
========================

🚫 Error "Puerto ocupado"
   - Cambiar puerto en vite.config.js
   - O terminar proceso: netstat -ano | findstr :5175

📱 No funciona en móvil
   - Verificar que estés usando la URL de red
   - Asegurar que ambos dispositivos estén en misma WiFi
   - Permitir certificado HTTPS en móvil

🎥 Cámara no funciona
   - Permitir permisos de cámara en navegador
   - Usar HTTPS (requerido para cámara)
   - Probar en Chrome/Edge (mejor compatibilidad)

🔄 Sincronización falla
   - Verificar que API esté corriendo (/api/orders)
   - Revisar consola del navegador para errores
   - Ambos dispositivos deben usar misma URL base
