⚠️ ¿PROBLEMAS?
==============

🔴 No instala dependencias:
   - Verificar Node.js instalado
   - Ejecutar: npm cache clean --force

🔴 No abre en móvil:
   - Usar URL Network (no localhost)
   - Verificar misma WiFi

🔴 Cámara no funciona:
   - Usar HTTPS (no HTTP)
   - Permitir permisos de cámara

🔴 Puerto ocupado:
   - Terminar otros servidores
   - O usar: npm run dev -- --port 3000
